OnlyBlock Help

Everything you need to know about using OnlyBlock

What is OnlyBlock?

OnlyBlock is a browser extension that automatically detects and blocks X.com (Twitter) profiles that contain OnlyFans links in their bio. It helps keep your timeline clean and focused by removing unwanted content.

How it Works

OnlyBlock scans X.com pages for user profiles and checks their bio text for common OnlyFans link patterns. When detected, the profile is automatically hidden from view.

Detected Link Patterns:

  • onlyfans.com
  • onlyfans.co
  • of.ly
  • fans.ly
  • Linktree pages containing OnlyFans
  • Bio.link pages containing OnlyFans

Blocked Phrases in Posts:

  • "my free of" (and similar variations)
  • Additional phrases can be added in the settings

Getting Started

  1. Install the Extension: OnlyBlock should already be installed in your browser.
  2. Enable Blocking: Click the OnlyBlock icon in your browser toolbar and toggle the switch to "ON".
  3. Visit X.com: Navigate to any X.com page to see OnlyBlock in action.
  4. Automatic Detection: Profiles with OnlyFans links will be automatically hidden.

Managing Blocked Profiles

You can view and manage all blocked profiles through the extension popup:

  • Click the OnlyBlock icon in your browser toolbar
  • View the list of blocked profiles
  • Click "Unblock" next to any profile to restore it
  • Toggle the extension on/off as needed

Settings & Customization

OnlyBlock is designed to work out of the box with sensible defaults. The extension automatically:

  • Saves your blocked profiles across browser sessions
  • Remembers your enable/disable preference
  • Works on both x.com and twitter.com domains
  • Updates in real-time as you browse

Troubleshooting

Extension Not Working?

  • Make sure the extension is enabled in the popup
  • Refresh the X.com page
  • Check that you're on x.com or twitter.com
  • Try disabling and re-enabling the extension

Profiles Not Being Blocked?

Note: OnlyBlock only detects profiles that are currently visible on the page. If you're not seeing blocked profiles, try scrolling to load more content or navigating to different pages.

False Positives?

Important: OnlyBlock uses pattern matching to detect OnlyFans links. In rare cases, legitimate content might be flagged. You can always unblock profiles manually through the popup.

Privacy & Data

OnlyBlock respects your privacy:

  • All data is stored locally in your browser
  • No data is sent to external servers
  • Blocked profiles are only stored in your browser's sync storage
  • You can clear all data by uninstalling the extension

Support

If you need help or have suggestions for improving OnlyBlock:

  • Check this help page for common issues
  • Review the extension settings
  • Try refreshing the page or restarting your browser
